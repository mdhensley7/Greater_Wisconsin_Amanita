About the Project

Greater Wisconsin Amanita

This project is designed to help naturalists, students, and mushroom enthusiasts identify Amanita species with a focus on Wisconsin and nearby regions. It combines structured search, local observation data, and species reference content in a simple, field-friendly interface.

How to Use the Application

The site is organized to help you move from trait-based search to local observations and species details.

  • Search: Enter morphological traits to narrow species candidates based on cap color, volva, stipe, and host tree.
  • Cross Reference: Compare multiple Amanita species using research-grade iNaturalist observations and local mapping data.
  • Species List: Browse species descriptions and species details as the directory grows.

Start with the Search page, then use Cross Reference to compare species and verify identification using nearby Wisconsin observations.

Project Context

Wisconsin hosts a diverse and important suite of Amanita species, some of which are toxic and others edible. This project is meant to be a trusted reference for those working in the field and for anyone who wants to learn more about local Amanita diversity.

It is built to be approachable, but also grounded in scientific observation data from iNaturalist and local mycological knowledge.
