# Greater_Wisconsin_Amanita

Welcome to the open source repository that maps out a hands-on mushroom identification tool!

The main objective of this project was to create a tool for people to make fine-grained species identifications within the genus Amanita in the upper midwestern region of the United States.

But that's a pretty niche subject right? So we hoped to make this a project that could be easily overhauled to any genus of mushroom in any part of the globe! 

If you are interested in building a similar tool please navigate to the markdown file for step-by-step instructions. Feel free to reach out to mdhensley7@gmail.com if you have any further questions.

Happy hunting!
