Welcome

This site will eventually host a fully interactive dichotomous key to species of the genus Amanita, including subgenera *Amanita*, *Lepidella*, *Amanitina*, and others. The goal is to create an accessible, accurate, and open tool for mycologists, students, and enthusiasts.

For now, this homepage serves as the foundational structure for the project. More components and features will be added as development continues.

Project Goals

  • Develop a clear, reliable dichotomous key for North American and global *Amanita* species.
  • Provide accessible morphological explanations with diagrams and photos.
  • Allow interactive branching choices (y/n, trait options, etc.).
  • Link each terminal choice to species descriptions and ecology.
